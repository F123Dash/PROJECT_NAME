1. NS-3 Style Architecture
    WHAT to implement
        1.1 Discrete Event Simulation Engine
            Core of your system.
                Event Queue (priority queue)
                Simulation Clock
                Event Scheduler
            Events you must support
                Packet arrival
                Packet transmission start
                Packet transmission end
                Queue enqueue/dequeue
                Timeout / retransmission
                Routing table update
                Link failure / recovery
        1.2 Node Abstraction
            Each router/node should have:
                ID
                Routing table
                Queue (buffer)
                Interfaces
                Protocol stack
        1.3 Link Model
            Each edge should include:
                bandwidth (bps)
                latency (ms)
                packet loss probability
                queue capacity
        1.4 Packet Model
                Packet {
                    source
                    destination
                    size
                    protocol (TCP/UDP/etc)
                    timestamp
                    TTL
                }


2. Performance Metrics (CRITICAL)
    2.1 Latency
        latency = arrival_time - send_time
        Track:
            per packet latency
            average latency
            worst-case latency
    2.2 Packet Loss
        Causes:
        buffer overflow
        link failure
        random loss
        Metric:
        loss rate = dropped / total_sent
    2.3 Throughput
        throughput = total_bits_received / time
    2.4 Bandwidth Utilization
        utilization = used_bandwidth / max_bandwidth
    2.5 Hop Count
        number of routers crossed
    2.6 Jitter (?????)
        variation in packet delay
    2.7 Queue Metrics
        queue length
        waiting time
        drop rate
    2.8 Routing Efficiency Metrics
        Compare:
            MST path vs shortest path
            Measure:
            cost difference
            path stretch
